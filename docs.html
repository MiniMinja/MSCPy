MMAT API Overview

GET /

Returns a basic system response to verify that the API is alive.

POST /room/

Create a new room.

  • r: (string) Name of the room

POST /broadcast/

Broadcast a message to a specific room.

  • r: (string) Room name
  • res: (string) Message to broadcast

GET /broadcast/

Retrieve all current broadcasts for a room.

  • r: (string) Room name

POST /results/

Post results or progress updates for a specific request.

  • r: (string) Room name
  • ID: (string) Request ID
  • res: (string) Either "PROGRESS" or "RESULTS"
  • body: (JSON) Object with keys progress and/or computedResult

GET /results/

Retrieve result for a specific ID in a room.

  • r: (string) Room name
  • ID: (string) Request ID

POST /close/

Close a room.

  • r: (string) Room name

GET /status

Get server-side status of room and process state.

POST /restart

Restart the backend communication process.
